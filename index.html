1. Mengapa menggunakan HTML, CSS, serta JavaScript dalam pemrograman web?

HTML (HyperText Markup Language): Digunakan untuk membuat struktur dasar halaman web. HTML mendefinisikan elemen-elemen seperti teks, gambar, tabel, formulir, dan tautan. Dengan kata lain, HTML adalah kerangka dari sebuah halaman web.

CSS (Cascading Style Sheets): Berfungsi untuk mendesain dan mempercantik tampilan web. Dengan CSS, pengembang dapat mengatur warna, ukuran, layout, dan animasi sehingga web menjadi lebih menarik dan responsif.

JavaScript: Digunakan untuk membuat halaman web lebih dinamis dan interaktif. Dengan JavaScript, pengembang dapat menambahkan fitur seperti validasi formulir, animasi, interaksi pengguna, dan komunikasi dengan server tanpa perlu memuat ulang halaman.

2. Bagaimana HTML, CSS serta JavaScript dapat diintegrasikan dalam pemrograman web?

HTML + CSS:

Internal CSS: CSS ditulis di dalam file HTML, tepatnya di dalam tag <style> di bagian <head>.

Eksternal CSS: CSS ditulis dalam file terpisah (.css) dan dihubungkan ke HTML menggunakan tag <link>.

HTML + JavaScript:

Internal JavaScript: JavaScript ditulis langsung di dalam file HTML, dalam tag <script>.

Eksternal JavaScript: Kode JavaScript disimpan di file .js terpisah dan dihubungkan ke HTML menggunakan tag <script src="nama_file.js">.
